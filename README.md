# JAZZ.EXE 🎰

> Builder. Ecommercer. KI-Nerd. Professioneller Glücksspieler.

Eine persönliche Spaß-Website im Retro-Terminal-Stil.

## 🚀 Live schalten (GitHub Pages)

1. Dieses Repository forken oder die Datei hochladen
2. **Settings** → **Pages** → Branch: `main` → Ordner: `/ (root)` → **Save**
3. Fertig — live unter `https://DEIN_USERNAME.github.io/REPO_NAME`

## 📁 Dateien

```
/
├── index.html      ← Die komplette Website (single file)
└── README.md       ← Diese Datei
```

Alles ist in einer einzigen `index.html` — kein Build-Prozess, keine Dependencies, kein Bullshit.

## 🎮 Features

- Boot-Sequenz beim Laden
- Matrix-Regen im Hintergrund
- Interaktives Terminal mit echten Commands
- Willkommens-Chat-Popup
- Seiten-Tutorial
- Skill Tree mit Animationen
- Projekt-Cards
- Easter Eggs (find sie)
- Vollständig responsiv (Desktop + Mobile)
- CRT-Scanlines & Glitch-Effekte

## 🔎 Easter Eggs

Es gibt welche. Such sie selbst. Hint: Terminal, ASCII-Logo, Stats-Box.

## ✏️ Anpassen

Alle Inhalte sind im `<script>` Block am Ende der `index.html`:

- `skillData` — deine Skills
- `projectData` — deine Projekte  
- `commands` — Terminal-Commands
- `bootMsgs` — Boot-Nachrichten
- Hero-Texte in `renderHeroLines()`
